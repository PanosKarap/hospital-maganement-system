# 🏥Hospital Management System

**Αυτό είναι ένα σύστημα προσομοίωσης ενός κέντρου διαχείρισης λειτουργιών για ένα νοσοκομείο.**

## Οι ρόλοι και οι λειτουργίες του καθενός

* **ΔΙΑΧΕΙΡΙΣΤΗΣ**
    * Προβολή όλων των χρηστών (Admins/Doctors/Patients) και των στοιχείων τους (Username/Ονοματεπώνυμο/κλπ.).
    * Προσθήκη νέου χρήστη στο σύστημα.
    * Διαγραφή χρήστη (Εύρεση με βάση το username).
* **ΓΙΑΤΡΟΣ**
    * Προβολή των μελλοντικών ραντεβού και των λεπτομερειών τους (Ημερομηνία/Με ποιον ασθενή).
    * Ακύρωση ραντεβού.
    * Καταχώριση ημερομηνιών διαθεσιμότητας για κλείσιμο ραντεβού.
* **ΑΣΘΕΝΗΣ**
    * Προβολή των προσωπικών του στοιχείων.
    * Προβολή παρελθοντικών ραντεβού και των λεπτομερειών τους (Ημερομηνία/Με ποιον γιατρό).
    * Κλείσιμο ραντεβού με γιατρό (Εφόσον είναι διαθέσιμος την συγκεκριμένη ημερομηνία).
    * Ακύρωση ραντεβού.
